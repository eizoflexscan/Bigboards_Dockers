How to structure your dataset?

  • Each variable forms a column,
  • Each observation forms a row,
  • First row contains full variable names with no spaces and no special characters,
  • First column contains observation ID,
  • No row or column should remain fully empty.

    • How to code your variables?

  • Categorical variables (ordinal or nominal) include at least one character,
  • Missing value are encoded as 'NA',
  • Time variable respect the format 'YYYY/mm/dd'.
